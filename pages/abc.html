ABC Race is a game that is designed to test a user's typing ability. The game asks for the user's name. Then it will present a randomized arrangement of the English alphabet. The user is timed to see how fast they can type this randomized arrangement. If they are fast enough, their score will be saved as a highscore.

ABC Race was created for a school in Varanasi, India. The concept and implementation followed direct requests from students at the school and organization. The game has proven to improve students' typing ability and letter familiarization. To learn more about the school, please watch the short documentary I filmed during my time there!
